Grocery Manager Help Guide

Getting Started

Welcome to Grocery Manager! This app helps you plan your grocery shopping and track your spending.

All your data is saved in your browser and can be downloaded as files for backup or transfer to another device.

Adding Grocery Items

  1. Click the Add Item button
  2. Fill in the item details:
    • Category: Select from dropdown (e.g., Fruits et l�gumes)
    • Product Name: Enter the item name
    • Brand: Optional brand name
    • Size/Details: Package size or quantity details
    • Quantity: Number of items needed
    • Price: Price per item in CAD
  3. Click Add to save the item to your list

Managing Your Grocery List

Edit Items:

  • Click the ?? Edit button next to any item
  • Modify quantities, prices, or other details
  • Click Save to confirm changes

Mark as Picked Up:

  • Check the ? box when you pick up an item while shopping
  • This helps track what you've collected

Search and Filter:

  • Use the search box to find specific items
  • Toggle "Show only unpicked items" to focus on what's left

Delete Items:

  • Click the Delete button to remove items

?? Saving Your Work

Save to Browser Storage:

  • Click the ?? Save button
  • Your data is saved locally in your browser
  • It will persist even if you close the browser
  • Note: Browser data is device-specific and can be cleared

?? Creating a Summary Report

  1. After shopping, click ?? Summary Report
  2. Enter the following information:
    • Date: Date of shopping trip
    • Store: Which store you shopped at
    • Actual Cost: Total amount you actually spent
  3. The estimated cost is calculated automatically from your picked items
  4. Click Save Summary to store this record
  5. Summaries help you track spending patterns over time

?? Download / Backup Your Data

Download buttons save files to your Downloads folder:

  • ?? Download Data - Saves your grocery list as a JSON file
  • ?? Download Summaries - Saves your summaries as a JSON file
  • Files are named with your customer ID and date for easy identification
  • Keep these files safe as backups
  • Use them to transfer data to another device or browser

?? Upload Data on Another Device

Transfer your data to a new device or browser:

  1. Sign in to Grocery Manager on the new device
  2. Click ?? Upload Data button
  3. Select your previously downloaded grocery data JSON file
  4. Click ?? Upload Summaries button
  5. Select your previously downloaded summaries JSON file
  6. Data will be loaded automatically and saved to browser storage

Note: The upload buttons look for files in your Downloads folder by default.

?? Viewing Statistics

  1. Click ?? Statistics button
  2. View spending patterns by store:
    • Average estimated vs actual costs
    • Total spending per store
    • Cost differences (savings or overspending)
  3. Statistics are calculated from your saved summaries
  4. You can also upload specific summary files to analyze past data

?? Tips & Best Practices

  • ? Save regularly - Click ?? Save after making changes
  • ?? Download backups periodically for safekeeping
  • ?? Create summaries after every shopping trip for accurate statistics
  • ?? Use search to quickly find items in large lists
  • ?? Update quantities as you plan meals for the week
  • Clean up old items to keep your list manageable
  • ?? Upload previous data when switching devices or browsers

?? Troubleshooting

Data not appearing after reload?

  • Make sure you clicked the ?? Save button
  • Check if browser data/cookies are being cleared automatically
  • Download your data as a backup and re-upload if needed

Upload not working?

  • Ensure the file is in JSON format
  • Verify the file was downloaded from this application
  • Check that the file isn't corrupted or empty

Can't find downloaded files?

  • Check your browser's Downloads folder
  • File names include customer ID and timestamp for identification
  • Look for files named like: grocery-data-CUSTOMER123-20240315-120000.json
